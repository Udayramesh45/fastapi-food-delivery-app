# 🍕 FastAPI Food Delivery Backend

## 📌 Project Overview
This project is a backend system built using **FastAPI** that simulates a real-world **Food Delivery Application**.

Users can browse menu items, place orders, add items to a cart, checkout orders, and explore menu items using search, sorting, and pagination.

This project was developed as part of the **FastAPI Internship Final Project**.

---

## 🚀 Features

- RESTful API development using FastAPI
- Pydantic request body validation
- CRUD operations for menu items
- Cart management system
- Checkout workflow
- Search menu items
- Sort menu items
- Pagination support
- Swagger UI API testing

---

## 🛠 Tech Stack

- Python
- FastAPI
- Pydantic
- Uvicorn
- Swagger UI
